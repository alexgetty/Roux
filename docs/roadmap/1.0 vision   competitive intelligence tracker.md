# Competitive Intelligence Tracker

Auto-Researcher variant focused on competitors.

## Core Concept

Autonomous bot that continuously monitors competitors, updates the graph, surfaces changes. Knowledge about the competitive landscape that compounds over time.

## Monitored Sources

- Competitor websites and blogs
- Product changelog pages
- Press releases
- Job postings (signals priorities)
- Social media / Twitter
- App store updates
- Patent filings
- SEC filings (if public)

## Processing

1. Scheduled crawl of monitored sources
2. Detect changes from last crawl
3. Extract: new features, pricing changes, messaging shifts, hiring signals
4. Create/update competitor nodes
5. Link to own roadmap items ("Competitor X launched feature we're planning")
6. Alert on significant changes

## Node Structure

- **Competitor**: Company overview, last updated
- **Feature**: What they offer, when launched
- **Pricing**: Current pricing model
- **Signal**: Interpreted data point (hiring ML engineers = investing in AI)

## Queries Enabled

- "What has Competitor X launched in the last 6 months?"
- "Which competitors have features we don't?"
- "Pricing comparison across all tracked competitors"
- "Job posting trends in our space"

## Enables

- Stay informed without manual research
- Roadmap decisions informed by market
- Early warning on competitive moves
- Historical record of competitive landscape evolution
